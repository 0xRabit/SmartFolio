# CryptoFolio 图标设置指南

本文档说明如何在 Chrome Extension 中使用新生成的图标文件。

## 已生成的图标文件

所有图标文件已保存在 `src/assets/` 目录下：

| 文件名 | 尺寸 | 用途 | 文件大小 |
|--------|------|------|----------|
| `icon16.png` | 16x16px | 工具栏小图标（favicon） | 0.7 KB |
| `icon32.png` | 32x32px | 工具栏标准图标 | 1.7 KB |
| `icon48.png` | 48x48px | 扩展管理页面 | 3.0 KB |
| `icon128.png` | 128x128px | Chrome Web Store | 11.1 KB |

## 设计说明

当前使用的是**横向公文包图标（绿色版本）**，具有以下特点：

- **配色方案**：翠绿色 (#10B981) - 与插件主题色一致
- **设计元素**：公文包 + 文件夹 + 上升趋势图表
- **视觉风格**：粗线条、高对比度、透明背景
- **优化特点**：
  - 横向公文包造型，传达"投资组合管理"概念
  - 占据几乎整个画布，留白极少
  - 线条粗细 8-10px，确保小尺寸下清晰可辨
  - 透明背景，适应各种界面环境
  - 内含上升趋势箭头，直观传达"增长"理念

## 配置步骤

图标文件已经配置完成，`manifest.json` 中的配置如下：

```json
{
  "icons": {
    "16": "assets/icon16.png",
    "32": "assets/icon32.png",
    "48": "assets/icon48.png",
    "128": "assets/icon128.png"
  },
  "action": {
    "default_icon": {
      "16": "assets/icon16.png",
      "32": "assets/icon32.png",
      "48": "assets/icon48.png",
      "128": "assets/icon128.png"
    }
  }
}
```

## 重新构建和加载

1. 运行构建命令：
   ```bash
   npm run build
   ```

2. 在 Chrome 中重新加载插件：
   - 打开 `chrome://extensions/`
   - 找到 CryptoFolio 插件
   - 点击"重新加载"按钮

3. 新图标将立即生效

## 设计理念

这个图标设计专门针对 Chrome 扩展的使用场景进行了优化：

1. **高辨识度**：粗线条设计确保在 16x16px 的小尺寸下依然清晰可辨
2. **品牌一致性**：使用与插件 UI 相同的翠绿色主题
3. **语义清晰**：公文包 + 增长图表，直观传达"加密资产组合管理"的核心功能
4. **适应性强**：透明背景适配浅色和深色主题
5. **专业感**：公文包造型传递专业的财务管理形象

## 注意事项

- 所有图标文件均为 PNG 格式，支持透明背景
- 图标已针对各个尺寸进行优化，确保在不同场景下清晰显示
- 如需修改图标，请保持相同的文件名和路径结构
- webpack 配置已包含图标文件的自动复制规则
